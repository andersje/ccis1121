• for CentOS, download the rpm from http://www.c-corp.net/linux/centos/4/generic/RPMS/i386/tripwire-2.4.1.2-4.i386.rpm
  • open the rpm with software installer, and install it that way
  • /usr/sbin/tripwire-setup-keyfiles
  • cryptographic setup script will ask for site and local passphrases. Pick something good
  • /etc/tripwire/twpol.txt should be examined by hand, using vi, nano or gedit
  • tripwire --init 2>errors
  • if you get errors, examine the policy file and correct the error, then enter new policy with:
    twadmin --create-polfile /etc/tripwire/twpol.txt
