• Installing: rpm --install NAMEOFFILE
  • Removing: rpm --erase NAMEOFPACKAGE
  • List installed files: rpm -ql NAMEOFPACKAGE
  • package info: rpm -qi NAMEOFPACKAGE
  • list all installed packages: rpm -qa
  • Find out which package owns a file: rpm -qf filename
  • find out info on an uninstalled package: rpm -qip RPMFILE
  • find out which files will be installed: rpm -qlp RPMFILE
